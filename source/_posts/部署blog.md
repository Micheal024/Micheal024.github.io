1. 安装hexo
```
npm install hexo-cli -g
```
2. 初始化hexo
```
hexo init
```
3. 安装依赖
```
npm install
```
4. 本地预览
```
hexo server
```
5. 生成静态文件
```
hexo generate
```
6. 部署到github
```
hexo deploy
```
7. 配置文件
```
_config.yml
```
8. 添加文章
```
hexo new "My New Post"
```
9. 发布文章
```
hexo deploy
```
